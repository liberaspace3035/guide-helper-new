# ドキュメント一覧

このディレクトリには、プロジェクトのドキュメントが格納されています。

## 主要ドキュメント

### セットアップ・クイックスタート

- **[SETUP.md](SETUP.md)** - セットアップガイド（環境構築、データベース設定、サーバー起動）
- **[NGROK_SETUP.md](NGROK_SETUP.md)** - ngrokを使用した外部アクセス設定

### 実装状況

- **[IMPLEMENTATION_STATUS.md](IMPLEMENTATION_STATUS.md)** - 実装完了状況の概要
- **[ADMIN_FEATURES.md](ADMIN_FEATURES.md)** - 管理者機能の実装状況と使用方法

### API・設計

- **[API_DESIGN.md](API_DESIGN.md)** - API設計ドキュメント（全エンドポイント仕様）
- **[API_FLOW_APPROVE_GUIDE.md](API_FLOW_APPROVE_GUIDE.md)** - ガイド承認APIの詳細な流れ

### トラブルシューティング

- **[TROUBLESHOOTING.md](TROUBLESHOOTING.md)** - よくある問題と解決方法
- **[DATABASE.md](DATABASE.md)** - データベース確認方法、SQLクエリ、データ不整合の分析
- **[DEVELOPMENT_NOTES.md](DEVELOPMENT_NOTES.md)** - 開発中の注意点やベストプラクティス

### テスト

- **[TESTING_CHECKLIST.md](TESTING_CHECKLIST.md)** - 動作確認チェックリスト
- **[ACTION_ITEMS.md](ACTION_ITEMS.md)** - 動作確認後のアクションアイテム
